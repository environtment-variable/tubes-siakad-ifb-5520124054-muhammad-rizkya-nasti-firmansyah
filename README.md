# SIAKAD - Sistem Informasi Akademik

Tugas Besar Mata Kuliah Web II (IF53413)

## Deskripsi Aplikasi
Aplikasi Sistem Informasi Akademik (SIAKAD) ini adalah platform berbasis web yang dibangun menggunakan framework Laravel. Aplikasi ini berfungsi untuk mengelola operasional akademik secara terpusat, mencakup data dosen, mahasiswa, mata kuliah, jadwal perkuliahan, dan Kartu Rencana Studi (KRS).

## Fungsi Halaman
* **Login**: Halaman autentikasi untuk akses Admin dan Mahasiswa menggunakan Laravel Auth.
* **Dashboard**: Halaman utama yang menampilkan ringkasan informasi dan navigasi sistem.
* **Data Dosen**: Modul CRUD (Create, Read, Update, Delete) untuk mengelola data dosen pengajar.
* **Data Mahasiswa**: Modul CRUD untuk mengelola profil mahasiswa.
* **Data Mata Kuliah**: Modul CRUD untuk mengelola daftar mata kuliah beserta jumlah SKS.
* **Data Jadwal**: Modul CRUD untuk mengelola jadwal perkuliahan yang terhubung dengan Dosen, Mata Kuliah, Hari, Jam, dan Kelas.
* **KRS (Kartu Rencana Studi)**: Modul bagi mahasiswa untuk memilih mata kuliah (input) atau membatalkan pilihan (drop) dengan validasi batas maksimal 24 SKS.
